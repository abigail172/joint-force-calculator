# joint-force-calculator
open-source python tools for biomechanic education

##Features
- Calculate joint reaction forces during movement
- Compute muscle moments and torques
- Visualize force vectors and biomechanics
- Interactive Jupyter notebooks for education

## Installation
```bash
git clone git@github.com:abigail172/joint-force-calculator.git
cd joint-force-calculator
pip install -r requirements.txt
